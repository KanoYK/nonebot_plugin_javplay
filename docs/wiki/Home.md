# JavPlay Wiki

JavPlay 是一个 NoneBot2 插件，用于在 Jellyfin 中实现虚拟影片点播、本地缓存下载和单影片刷新。

推荐阅读顺序：

1. `01-Quick-Start.md`：从零部署顺序。
2. `02-Path-Mapping.md`：解释 NoneBot、Jellyfin、Aria2 三套路径如何对应。
3. `03-External-Tools.md`：FlareSolverr、Aria2、MetaTube Server、115 的部署说明。
4. `04-Configuration.md`：`.env` 配置项。
5. `05-Troubleshooting.md`：常见问题。

请不要把 Jellyfin API Key、115 Cookie、Webhook Token、账号密码或日志中的敏感 URL 发到公开仓库。
